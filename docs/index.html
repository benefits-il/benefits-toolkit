היסטוריית השיחות (Claude Chats)

סרגל צד פשוט שמציג את כל השיחות הקודמות שלך עם Claude Code. האייקון שלו מופיע בסרגל הצידי של VS Code ופותח את רשימת השיחות בלחיצה.

  • חיפוש וסינון: אפשר לחפש שיחות מהר או לסנן רק את השיחות של הפרויקט שאתה עובד עליו כרגע.

  • קיצור דרך תחתון: כפתור בסרגל הסטטוס התחתון מאפשר לתת שם לשיחה הנוכחית ברגע.

כיווניות ופונטים

יישור עברית ופונטים מותאמים בצ'אט

כותבים וקוראים עברית בצורה חלקה בתוך הטרמינל של Claude Code. הטקסט לא מתהפך, וסימני הפיסוק והקוד נשארים בדיוק במקום הנכון.

  • לבחור מתי להפעיל: מצב ידני עם כפתור החלפה (Active), מיושר תמיד (Always) או זיהוי שפה אוטומטי (Auto).
  • להתאים את המראה: אפשר לבחור פונט מותאם אישית לטקסט ופונט לקוד כדי שיהיה נוח בעין.

צליל קצר מושמע ברגע שקלאוד מסיים לעבוד או מחכה לתשובה שלך. ככה לא צריך לבהות במסך ולחכות לו כל הזמן.

USAGE & COMMANDS

איך משתמשים?

משנים הגדרות בקלות דרך הגדרות ה-VS Code או מריצים פקודות ישירות מלוח הפקודות.

הגדרות פשוטות (Settings)

כל ההגדרות נמצאות תחת השם benefit.*. אפשר להפעיל או לכבות כל פיצ'ר בנפרד:

benefit.rtl.enabled
להפעיל או לכבות את יישור העברית (RTL) בצ'אט.
benefit.rtl.mode
איך ליישר את הטקסט: "active" (כפתור ידני), "always" (תמיד מיושר), "auto" (זיהוי אוטומטי).
benefit.sounds.enabled
להפעיל או לכבות את צלילי ההתראה של קלאוד.
benefit.sounds.stopVariant
איזה צליל לשמוע כשמשימה מסתיימת: "default" או "alt".
benefit.chats.enabled
להפעיל או לכבות את סרגל ההיסטוריה בצד.
benefit.chats.scope
איזה שיחות להציג: רק של הפרויקט הפתוח ("currentWorkspace") או של כל הפרויקטים ("all").

פקודות מהירות (Commands)

פותחים את לוח הפקודות (קיצור Ctrl+Shift+P או Cmd+Shift+P) ומקלידים:

Benefit: RTL: Toggle
להדליק או לכבות עברית בצ'אט הנוכחי ברגע.
Benefit: Sounds: Install Hooks
להתקין ולחבר את צלילי ההתראה ל-Claude Code במחשב.
Benefit: Sounds: Choose Sound Variants
לבחור איזה סוגי צלילים יושמעו.
Benefit: Sounds: Test Playback
לנגן צליל בדיקה כדי לוודא ששומעים אותו.
Benefit Chats: Search Conversations
לחפש שיחה ישנה במהירות מתוך כל ההיסטוריה.
Benefit Chats: Rename Active Conversation
לתת שם לשיחה הנוכחית כדי למצוא אותה בקלות אחר כך.
PRODUCTIVITY & VALUE

מה זה נותן לנו?

איך Benefit's Toolkit עושה לכם חיים קלים ועוזר לכם לעבוד הרבה יותר מהר.

עבודה שוטפת ונוחה בעברית

בלי להיאבק עם סימני פיסוק הפוכים או טקסט מבולגן. פשוט כותבים ומקבלים תשובות בעברית תקינה ובצורה קריאה.

עבודה בלי להיתקע מול המסך

במקום לשבת ולחכות שקלאוד יסיים להריץ פקודות או יבקש מכם אישור, אתם יכולים לעבור לטאב אחר. צליל קצר יקרא לכם ברגע שצריך אתכם.

למצוא הכל ברגע

בלי להתחיל שיחות מחדש מאפס או לשמור קוד בצד. בקליק אחד חוזרים לשיחה מאתמול, מחפשים פתרונות ישנים או מייצאים את השיחה לקובץ Markdown.
