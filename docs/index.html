INSTALL

התקנה

  1. הורד את הקובץ benefits-toolkit-x.x.x.vsix מעמוד ה-Releases.
  2. ב-VS Code: לוח Extensions ← תפריט (···) ← Install from VSIX… ← בחר את הקובץ שהורדת.
  3. או דרך הטרמינל:
    code --install-extension benefits-toolkit-0.1.1.vsix
  4. הפעל מחדש את VS Code. סרגל ה-Benefit יופיע בצד.

גם ב-Cursor: אותו קובץ VSIX עובד — Cursor תומך ב-Install from VSIX בדיוק כמו VS Code.
