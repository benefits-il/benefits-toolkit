FEATURES

שלושה כלים, עבודה אחת.

כל אחד פותר חיכוך אמיתי בעבודה היומית מול Claude Code — בלי הגדרות, בלי תלויות.

עברית · RTL

תמיכת כיווניות מלאה בצ'אט של Claude Code: טקסט עברי מיושר נכון, פיסוק במקום, וערבוב עברית-אנגלית קריא — בלי הקפיצות המעצבנות.

צלילי התראה

Hooks קוליים שמודיעים מה קורה: צליל כשמשימה מסתיימת, כשצריך אישור, וכשיש שגיאה. תפסיק לבהות במסך ולחכות.

היסטוריית שיחות

סרגל צד שמרכז את כל השיחות שלך: דפדוף, חיפוש, וחזרה לכל שיחה קודמת בלחיצה — בלי לאבד הקשר.
